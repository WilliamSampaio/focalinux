help

Ajuda rápida, útil para saber que opções podem ser usadas com os comandos internos do interpretador de comandos. O comando help somente mostra a ajuda para comandos internos, para ter uma ajuda similar para comandos externos, veja “Ajuda na própria linha de comandos”. Para usar o help digite:

help [comando]

Por exemplo, help echo, help exit
