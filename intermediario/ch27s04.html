Arquivo /etc/console-tools/config

Este arquivo contém configurações padrões do pacote para as fontes de tela e mapas de teclado usados pelo sistema. A fonte de tela é especificada neste arquivo (as fontes disponíveis no sistema estão localizadas em /usr/share/consolefonts).

Os arquivos de mapa de teclados estão localizados no diretório /usr/share/keymaps/.
